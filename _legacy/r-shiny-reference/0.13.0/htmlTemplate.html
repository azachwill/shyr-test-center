Process an HTML template

htmlTemplate(filename, ..., document_ = "auto")

Arguments

filename Path to an HTML template file.
... Variable values to use when processing the template.
document_ Is this template a complete HTML document (TRUE), or a fragment of HTML that is to be inserted into an HTML document (FALSE)? With "auto" (the default), auto-detect by searching for the string "<HTML>" within the template.

Description

Process an HTML template and return a tagList object. If the template is a complete HTML document, then the returned object will also have class html_document, and can be passed to the function renderDocument to get the final HTML text.
