Create a list of tags

Source: R/tags.R

Description

Create a list() of tags with methods for print(), as.character(), etc.

tagList(...)

Arguments

...

A collection of tags.

Examples

tagList(
  h1("Title"),
  h2("Header text"),
  p("Text here")
)
